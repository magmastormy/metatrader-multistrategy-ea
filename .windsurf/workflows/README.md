# Windsurf Workflows For This Repository

Use these from Windsurf with `/` and select the workflow command.

## Available workflows
- `/audit_runtime_ai`: Full runtime + code-path audit for AI/strategy execution issues.
- `/implement_safe_fix`: Implement a change with compile/test/doc safeguards.
- `/shadow_run_triage`: Run shadow-mode validation and extract actionable diagnostics.
- `/release_doc_sync`: Finalize docs/changelog after implementation.
- `/compile_clean`: Compile with artifact hygiene rules.
- `/global_task_loop`: Universal workflow for most coding tasks.
- `/web_build_adjust_debug`: Build websites/web apps and handle iterative UI/behavior adjustments.
- `/web_debug_hotfix`: Fast web regression debugging and production-safe fixes.
- `/python_feature_bugfix`: Python feature delivery with tests, lint, and safe refactor flow.
- `/python_debug_triage`: Python runtime/debug workflow for crashes, logic bugs, and flaky tests.

## Notes
- Files in subfolders are also discovered by Windsurf.
- Keep workflow names stable so team habits and slash commands remain consistent.
